1. `git clone git@github.com:FactomProject/factom-node-monitoring-tool.git`
2. `git checkout Clay`
3. `cd factom-node-monitoring-tool`
4. `npm install`
5. `node server.js`
   _IN A SECOND TAB_
6. `npm run start`
   ^^^ should open a tab up for you.

in the `.env` file There are instructions on how to the arrays format.
